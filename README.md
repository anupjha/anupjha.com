## My Blogs


### Tech Stack
- NextJS 14 basics (layouts, app router etc)
- NextJS SEO
- NextJS Dynamic Graph Images
- [Velite](https://velite.js.org/) Setup and usage
- [Shadcn/ui](https://ui.shadcn.com/) setup and usage
- Custom components in MDX
- Tailwind styling


### Install

```pnpm install``

```
pnpm dev
pnpm build
pnpm start
pnpm lint

```
